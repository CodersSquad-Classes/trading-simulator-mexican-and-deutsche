## How to build and run

### Linux / macOS / WSL
1. Compile: `g++ -std=c++11 main.cpp MatchingEngine.cpp Display.cpp -o market`
2. Run: `./market`

### Windows (MinGW)
1. Compile: `g++ -std=c++11 main.cpp MatchingEngine.cpp Display.cpp -o market.exe`
2. Run: `market.exe`
